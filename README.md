# рџљЂ AI Push-Up Counter

### Real-Time Workout Rep Detection using FastAPI, MediaPipe & Computer Vision

AI-powered push-up counting system that detects human body pose from webcam or video, counts repetitions in real time, analyzes workout form, and provides performance insights through an API and lightweight dashboard.

Built with **FastAPI, OpenCV, MediaPipe, and Python**, this project demonstrates how computer vision and AI can be applied to fitness tracking and smart workout assistants.

---

# рџ“Њ Project Overview

Traditional workout tracking requires manual counting or expensive fitness devices.
This project uses **computer vision and pose estimation** to automatically detect and count push-ups while analyzing exercise form.

The system processes webcam or video input, detects human pose landmarks, calculates joint angles, and determines push-up repetitions through a motion state machine.

The backend API exposes workout data and analytics for dashboards or fitness applications.
